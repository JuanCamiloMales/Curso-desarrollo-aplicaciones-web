Enlaces relativos y absolutos

Enlaces absolutos

Las URL absolutas incluyen todas las partes de la URL (protocolo, servidor y ruta) por lo que no se necesita más información para obtener el recurso enlazado.

Ejemplos enlaces absolutos

https://www.w3.org/

https://www.google.com/

Enlaces relativos

las URL relativas prescinden de algunas partes de las URL. Como se trata de URL incompletas, es necesario disponer de información adicional para obtener el recurso enlazado. En concreto, para que una URL relativa sea útil es imprescindible conocer la URL del origen del enlace. Las URL relativas se construyen a partir de las URL absolutas y prescinden de la parte del protocolo, del nombre del servidor e incluso de parte o toda la ruta del recurso enlazado..
